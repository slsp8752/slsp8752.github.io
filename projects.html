CU Grand Challenge - Aeriel Photogrametry

As a part of CU-Boulder's 2015 Grand Challenge, I helped create a system to demonstrate the use of UAVs to map terrain. People who participated in our exhibit "flew" a camera over a scale model of Tikal, Guatemala, and then had a 3D model made from their photos using Agisoft Photoscan. You can check out the github project page here.

Dinotify

Dinotify is a system for communicating with the ESP8266 wifi module to create simple internet connected objects. The name comes from its initial purpose of creating a toy dinosaur which lights up when you receive an email. It is currently in the early stages, but you can track my progress here.
